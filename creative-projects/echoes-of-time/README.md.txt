# ⏳ Echoes of Time

A text-based adventure game built with Python where players travel through different time periods to recover the Time Crystals and restore the timeline.

## Story

Guided by the AI assistant **Aurabee**, you explore Ancient Egypt, the Medieval Kingdom, and the Future City using the mysterious **Time Tube** created by **Dr. A. Jobateh**.

## Features

* Interactive story
* Multiple chapters
* Riddles and logic puzzles
* Random challenges
* Lives and Time Crystal tracking

## Skills Used

* Functions
* Lists
* Dictionaries
* Loops
* Conditionals
* Random module
* User input
